# UM Loop Status

Check the current state of the UM Loop:

1. **Read PRD.md** if it exists
2. **Read progress.md** if it exists
3. **Report**:
   - Total tasks (count checkboxes)
   - Completed (count `[x]`)
   - Remaining (count `[ ]`)
   - Last progress entry
   - Next item to work on

If no PRD.md exists, report: "No active UM Loop. Start one with `/um-loop start [task]`"
